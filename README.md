# Cycle Time Histogram

![Version](https://img.shields.io/badge/version-1.2.0-blue)

![Sample](images/sample.jpg)

## What is this

The purpose of this widget is to show a Cycle Time Histogram based on a selected Shared Query. Also you could define percentiles to be shown on the chart.

## Configuration

To configure this widget you must select a Shared Query and two field dates.

![Configuration](images/configuration.jpg)

## Development

```bash
npm install
npm run build
npm run package
npm run publish:dev
```
